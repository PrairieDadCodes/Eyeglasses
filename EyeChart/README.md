# EyeChart

EyeChart is a tool to build TGM4 replay files.

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/prairiedadcodes/Eyeglasses.git
   cd Eyeglasses/EyeChart
   ```

2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

## Usage

Run the application:
```
python eyechart
```

## Dependencies

- Python 3.8+
- construct 2.10+
